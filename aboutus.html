IGDTUW Campus Map A fully interactive and immersive 3D campus map for Indira Gandhi Delhi Technical University for Women (IGDTUW), designed to offer a realistic and detailed virtual tour experience.

Technologies Used

  • Frontend :HTML, CSS, JavaScript
  • 3D Rendering :Three.js, WebGL
  • 3D Modeling :Blender
  • Animations :GSAP (GreenSock Animation Platform)
  • Development Tools :Visual Studio Code, GitHub, npm

Features

Interactive 3D Model

Explore the entire campus in a 3D environment, with easy navigation using mouse or keyboard controls.

Real-Time Rendering

Real-time rendering allows users to view the campus map with high-quality visuals and smooth transitions.

Floor Navigation

Quickly switch between different floors of the campus, making it easy to locate buildings, rooms, and facilities.

Building Details

Click on individual buildings to view detailed information, including department names and office numbers.

Interactive Campus Points

Click on specific areas for pop-up information about amenities, cafeterias, and other important spots on campus.

Real-World Visuals

High-quality textures and models represent the campus with a high level of accuracy, creating a real-world experience.
